Welcome to ES2015 Time Manager

You can start/stop by ENTER or SPACE key

You can MANUALLY enter the value in field

All your settings are STORED in the browser

The current value and the name of the counter is displayed in the TITLE of the browser tab

You can select a MELODY , the VOLUME or MUTE the ringer

There are 3 MODES :

  • TIMER (count down)
  • STOP WATCH (count up)
  • WATCH (clock)

Timer also can count up after the end of time (customizable )

Enjoy using!
